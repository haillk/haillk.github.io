Golang生成汇编代码

2020年5月9日
| 未分组
| 阅读

佛门广大，岂不容一癫僧！

1、go tool compile

go tool compile -N -l -S main.go

go tool compile -help获取细节

-N 关闭优化

-l 关闭内联

-S 打印组件清单

2、go tool objdump

首先编译程序go tool compile -N -l main.go

使用go tool objdump main.o反汇编出代码。

3、go build -gcflag -S

go build -gcflags -S main.go

总结

1和3生成的是过程中的汇编，最终的机器码的汇编可以用2生成。
