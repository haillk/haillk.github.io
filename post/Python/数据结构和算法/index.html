序列元组赋值

​ 元组或者序列中的值，可以直接赋给变量，但是数量要相等。

​ 字符串也可以直接赋值。

​ 对于不需要的值，可以用空变量获取，然后舍弃不要。

Collection.deque() 返回一个两头队列deque(double ended queue)

可以用来保留最后N和元素

实例：

from collections import deque

q=deque(maxlen=3)
q.append(1)
q.append(2)
q.append(3)
q.append(4)
print(q)
