线程锁

1

set scheduler-locking off|on

off:不锁定任何线程，当程序继续运行的时候如果有断点，那么就把所有的线程都暂停下来，直到你指定某个线程继续执行，如果在当前线程下使用continue的话那么会启动所有线程（GDB默认）； on:打开线程锁，锁定其他线程，只有当前线程执行。

non-stop模式

当某个或多个线程在一个断点上，其他线程仍会并行运行 \2. 你可以选择某个被中断的线程，只让他运行。

set target-async 1
set pagination off
set non-stop on
